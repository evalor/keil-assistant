# 项目简介

## 项目概述

**Keil Assistant** 是一个 VS Code 扩展插件，为 Keil uVision 5+ 项目提供完整的集成开发体验。该插件弥合了现代代码编辑器与传统嵌入式开发工具链之间的鸿沟。

## 核心目标

### 主要目标
1. **无缝集成**：在 VS Code 中直接操作 Keil 项目，无需频繁切换工具
2. **智能化构建**：自动检测项目类型、优化编译参数、提供实时反馈
3. **开发效率**：通过智能提示、快捷操作和状态可视化提升开发体验
4. **跨项目支持**：同时管理多个 Keil 项目，支持 C51/C251/ARM 架构

### 解决的问题
- Keil uVision IDE 界面陈旧，缺乏现代编辑器特性
- 项目文件管理复杂，难以快速定位和编辑
- 编译过程不透明，错误信息难以追踪
- 多项目切换繁琐，缺乏统一管理界面
- IntelliSense 配置复杂，代码提示不准确

## 项目范围

### 支持的功能
- ✅ 加载和显示 Keil 项目结构（C51/C251/ARM）
- ✅ 一键编译、重新编译、下载程序
- ✅ 自动生成 C/C++ IntelliSense 配置
- ✅ 实时监测项目文件变化
- ✅ 多核心并行编译优化
- ✅ 智能错误诊断和定位
- ✅ 项目目标切换
- ✅ 状态栏快捷操作

### 技术约束
- **平台限制**：仅支持 Windows（Keil uVision 是 Windows 专属）
- **Keil 版本**：需要 Keil uVision 5 或更高版本
- **VS Code 版本**：需要 VS Code 1.85.0+
- **依赖要求**：需要安装 C/C++ 扩展以获得完整功能

### 不在范围内
- ❌ Keil uVision 的替代品（仅提供集成，不替代 Keil）
- ❌ 调试功能（使用 Keil 或其他调试工具）
- ❌ 跨平台支持（macOS/Linux）
- ❌ 非 Keil 工具链支持
